# Evaluation Reports — GitLab Pages

This repository hosts self-contained HTML evaluation reports generated by
[prototype-creator](https://github.com/andybraren/prototype-creator).

Reports are deployed automatically via GitLab Pages on push to `main`.
Internal-only — requires Red Hat VPN + GitLab auth.

## Structure

```
public/
└── evals/
    ├── index.html              # Browsable report index
    ├── RHAISTRAT-1536/
    │   └── index.html          # Eval report for RHAISTRAT-1536
    └── ...
```

## Publishing

Reports are published by `scripts/publish-report.sh` in the prototype-creator
repo. Re-running an eval replaces the report at the same URL.

## Access

Internal-only — requires Red Hat VPN and GitLab authentication.
